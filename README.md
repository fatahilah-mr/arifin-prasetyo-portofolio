# Arifin Prasetyo - Premium Culinary Portfolio

Sebuah website portofolio kuliner premium yang dirancang khusus untuk **Arifin Prasetyo**, seorang **Professional Cook** dan **Lulusan Terbaik Jurusan Kuliner 2026**. Proyek ini dibangun menggunakan arsitektur modern berbasis komponen modular, mengutamakan pendekatan **mobile-first**, serta dioptimalkan untuk performa dan indeksasi mesin pencari.

🌐 **Live Demo:** <https://arifin.fatahmr.my.id>

---

## 🚀 Fitur Utama

- **Floating Navigation Bar** — Navbar modern dengan efek *backdrop blur* dan menu responsif.
- **Dynamic Culinary Gallery** — Sistem filter berbasis state untuk 14 hidangan (*Main Course*, *Pastry & Bakery*, dan *Beverages*).
- **Responsive Credentials Timeline** — Timeline sertifikasi profesional dengan rasio gambar yang responsif.
- **Direct Call-to-Action (CTA)** — Tombol menuju WhatsApp dan Email resmi Arifin.
- **Production-Ready SEO Config** — Mendukung Cloudflare Pages, Google Search Console, `robots.txt`, `sitemap.xml`, dan `_redirects`.

---

## 🛠️ Tech Stack

- **Framework & Build Tool:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Architecture:** Lovable modular tier architecture
- **Deployment:** Cloudflare Pages

```text
arifin-prasetyo-portofolio/

```

## 💻 Pengembangan Lokal

### 1. Clone repositori
```bash
git clone https://github.com/fatahilah-mr/arifin-prasetyo-portofolio.git
cd arifin-prasetyo-portofolio
```
### 2. Instal dependensi menggunakan Bun
```bash
bun install
```
### 3. Jalankan server pengembangan
```bash
bun run dev
```
